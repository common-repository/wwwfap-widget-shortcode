Plugin to enable shortcode in text widgets and custom html widgets on your WordPress site.

== Description ==
A free to use WordPress plugin that will allow you to run shortcode within widgets. 
Works in standard “text” and “custom html” widgets. 
Extremely light weight, just a few lines of code. Install and activate, there are no settings, it simply works. 
 
== Installation ==

* Upload the plugin to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress

Once you have completed these steps you will be able to use shortcode in your widgets. Enjoy!
 
== Frequently Asked Questions ==
 
= Are there any settings that I need to configure? =
 
Nope. Just activate the plugin. That's all.
